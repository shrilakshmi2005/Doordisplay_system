Arduino-Based Door Display System

 Overview
The Arduino-Based Door Display System is an embedded system that automatically detects the presence of a visitor using an HC-SR04 ultrasonic sensor and an IR obstacle sensor. When a visitor is detected, the system displays a welcome message on a 16×2 LCD and turns ON an LED. If no visitor is detected, the LCD displays "HAVE A GOOD DAY".

Features
- Automatic visitor detection
- LCD message display
- LED indication
- Ultrasonic distance sensing
- IR obstacle detection

Components Used
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- IR Obstacle Sensor
- 16×2 LCD Display
- LED
- Breadboard
- Jumper Wires

Working
The Arduino continuously reads the ultrasonic and IR sensors. When a visitor is detected, the LCD displays "WELCOME"and the LED turns ON. Otherwise, the LCD displays "HAVE A GOOD DAY" and the LED remains OFF.
